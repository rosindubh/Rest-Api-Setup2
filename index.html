REST API SET-UP INSTRUCTIONS

Introduction

          This is a step by step guide to create a REST-API 
with a connection to a MongoDb Database.

Subjects covered will include

  • Create a local server
  • Making a connection to a MongoDb
  • Writing and testing the controllers
  • Writing the Routes
  • Writing the Model
  • Test we have a full CRUD API

First Step

          First step is to setup the folders and files reqired.
        Make a Bash script called boilerplate.sh that looks
        like this.
        

>.env
>.gitignore
mkdir src
mkdir src/controllers
mkdir src/db
mkdir src/middleware
mkdir src/models
mkdir src/routes
mkdir src/user
>src/server.js
>src/controllers/index.js
>src/db/connection.js
>src/middleware/index.js
>src/models/index.js
>src/routes/index.js
>src/user/index.js


Then execute the script to create 
files and folders.
